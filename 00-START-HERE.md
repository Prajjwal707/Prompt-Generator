## 🔧 Easy to Customize

### Colors
Change in `tailwind.config.js`:
```js

```

### Navigation
Edit in `src/components/Sidebar.jsx`

### Pages
Add in `src/pages/` then route in `AppRouter.jsx`

### Components
Modify any file in `src/components/`
